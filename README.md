# DevPath

It is a calm mentor that quietly helps students find their path.

In other words, a personalized career navigation platform that helps Computer Science students make confident learning decisions through adaptive learning journeys based on their skills, goals, and progress.

Instead of following generic roadmaps or scattered resources, DevPath helps students understand where they currently stand, choose what to learn next, and track their progress through a structured learning journey.

It states that you don't need to learn everything.
You only need to know
what comes next.
